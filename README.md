# Homebrew Tap for Monokit

Teletype-style scripting for a SuperCollider complex oscillator voice.

## Installation

```bash
brew tap stolmine/monokit
brew install monokit
```

## Requirements

- macOS
- SuperCollider (installed automatically via Homebrew)
